# Login Error Message Bug

## Bug ID
BUG-001

## Summary
Error message is not properly aligned on invalid login attempt.

## Environment
- Browser: Chrome
- OS: Windows 11
- Website: SauceDemo

## Steps to Reproduce
1. Open login page
2. Enter valid username
3. Enter invalid password
4. Click Login button

## Expected Result
Error message should appear properly aligned and readable.

## Actual Result
Error message overlaps with login form layout.

## Severity
Low

## Priority
Medium
